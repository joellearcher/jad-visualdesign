The images were opened in Photoshop. For each image I duplicated a layer to make sure I had a copy of the original.
I then added an adjustement layer to increase the contrast, exposure, vibrance and saturation.
The original images were quite flat and so I did this to give the images more clarity and brighter colours, especially as blues,
yellows and orange are synonymous with holidays in the Caribbean islands.
The contrast was increased to adjust the lighter and darker areas, this was doen to make the image pop.
The exposure was used to adjust the highlights for more dynamic range.
The vibrance and saturation has been used to increase the vividness of the sand and water.
All three of the final images were exported using the Save for Web option in Photoshop.
I made sure to keep the image size to less than 300KB by saving in the high quality JPEG option,
which will make loading the website much faster for most users.
I wanted my island images to look more of a square size as its because of mobile viewing.
For the purpose of this assignment I have resized the before image so that you can see the
images well I have included the images before they were edited to show the manipulation.

Image 4 - Website Logo

Final image size: width 250px x height 250px (scaled down to width 80px x height 80px for the purpose of this assignment and website), JPEG, 5KB.

This is the logo for my website. I created a canvas size of 1080px x 1080px in Photoshop.
The font chosen for the text type is Bebas Neue which I think works well as I wanted to created a strong and bold looking logo.
The purple colour chosen was checked for contrast ratio using a website for accessibility.
I used the rulers and snap to options in Photoshop to make sure the text was placed in the center of the canvas.
I exported the logo for the web in PNG-24 format with a size of width 250px x height 250px, scaled down to width 80px x height 80px on the actual website.
I chose this size because I wanted to fit the logo within the navigation menu in the header.

I have used this dice image in various places around the website.
I wanted to change the original grey colour of the dice to match the colours chosen for elsewhere on the website.
The image was opened in Photoshop and I right-clicked on the layer and selected Blending Options.
I then clicked on Color Overlay, selected the Color Picker and chose the purple colour I have used elsewhere.
To create the small circles on the dice I have used the Ellipse tool and coloured them in a very light grey.
I have created six layers of the circles to show the different numbered faces of the dice.
The image shown below is just one of six images used for the dice game.

Image 6 - Social Media Image Sprite

Final image size: width 175px x height 43px, PNG, 12KB

This image sprite was created to allow users to share links from the website easily using different social media applications.
The image sprite was the best option to be able to keep the file size as small as possible.
To create the image sprite I used Photoshop. I clicked on File, Scripts and Load Files into Stack and then selected the 4 PNG files from my folder.
Because I used four PNG images I selected Image, Canvas Size and then change the width to 550.
Each PNG image was 128px x 128pw and so I times the width by four and then added 38px to allow me to space out the logos so that they were not touching each other.
The height was kept at 128px.
I made sure the Anchor was relative to the left-hand side.
I exported the image as one PNG file.
